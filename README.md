# Lab10
My code for the 10th lab of beginner's java, which takes user input for students' names and grades and provides a message that compiles each set of information.

## How to Run
1. Open the program in your IDE of choice.
2. Enter student names and scores when prompted.

## Example Output
How Many Students? 2
Enter Student Name: Jeffery
Enter Score: 50
Jeffery scored 50
Enter Student Name: Johnny
Enter Score: 90
Johnny scored 90
